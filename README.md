# README


### Ruby version
    Ruby 2.5.1

### Configuration
Run the following command in your terminal:
`bundle install`

Create application.yml file inside config folder and add following configuration values:
- DB_USER_NAME: '********'
- DB_PASSWORD: '********'
- TEST_DB_NAME: '********'
- DB_NAME: '********'

### Database creation
Run the following command in your terminal:
`rake db:create`

### Database initialization
Run the following command in your terminal:
`rake db:migrate`

### Database initialization
Run the following command in your terminal:
`rspec`
